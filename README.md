# Fast & Compact 3D-CNN for HSI-Classification

## Project Description
This repository contains the implementation of a fast and compact 3-D CNN designed for classifying HSI. The model has been tested on three geographically diverse 
## Datasets:

### Pavia University Dataset (PUD)
- **Location**: Italy
- **Ground Truth Groups**: 9
- **Spatial Dimensions**: 610x610
- **Spectral Bands**: 103
- **Description**: Collected using an advanced camera that can distinguish between different light sources.

### Salinas Dataset (SD)
- **Location**: California, USA
- **Classes**: 16
- **Spatial Dimensions**: 512x217
- **Spectral Dimensions**: 224
- **Description**: Gathered using a specialized camera designed for hyperspectral imaging.

### Indian Pines Dataset (IPD)
- **Location**: Indiana, USA
- **Classes**: 16
- **Spatial Dimensions**: 145x145
- **Spectral Dimensions**: 224
- **Description**: Collected using a camera capable of capturing both visible and infrared light.

## Evaluation Metrics
The model's performance is evaluated using the following metrics:
- F1-score
- Precision
- Recall rate
